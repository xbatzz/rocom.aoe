# 变更记录

## 2026-07-08

- 重排 `/pvp-lite` 为手机游戏助手式主界面，强化双方对位舞台、主结果卡和最高推荐技能卡，辅助信息与详细计算下沉。
- 新增 `/pvp-lite` 移动端轻量对战助手，复用当前队伍、属性倍率、实战速度线、纸面伤害和联防候选，并在首页与侧边栏新增入口，原 `/pvp` 详细版保留。
- 修复 `/pvp` 纸面伤害无法读取当前队伍构筑技能的问题，兼容 `/team` 保存的 `moveIds` 并通过 `PetSkillIndex` 映射宠物技能 ID。
- 修正 `/pvp` 纸面伤害公式口径：默认等级改为 60，本系倍率改为 1.25，并将显示威力与预计伤害取整顺序对齐 rocopvp battle-use-guide。
- 新增 `src/lib/damageCalculator.ts`，复用 `statCalculator` 的实战属性面板，提供固定威力物理 / 魔法攻击技能的纸面伤害估算。
- `/pvp` 新增纸面伤害计算 v0.1，只支持我方技能打对方，优先读取当前激活队伍槽位保存的技能、性格和个体值，并排除状态、防御、无固定属性和无固定威力技能。

## 2026-07-07

- `/pvp` 属性倍率卡片新增我方抗性候选功能，点击对方属性后默认读取当前激活队伍并展示 0.5x / 0.25x 联防参考。
- 配队页新增多配队管理 v0.1，支持新建、切换、重命名、复制和删除队伍，并将 `teamStorage` 升级到 `rocom.team-builder.v2` 多队伍结构，兼容旧 `rocom.team-builder.v1` 单队伍数据；`/pvp` 继续通过 helper 读取当前激活队伍。
- 调整 `/pvp` 和 `/stats` 宠物选择器默认展示逻辑，无搜索词时不再显示编号前 50，改为搜索驱动并保留当前已选展示。
- 优化 `/pvp` 和 `/stats` 宠物选择器体验，搜索选择后不再跳回默认前列，并新增当前已选展示、候选高亮和空状态。
- `/pvp` 对方速度区域新增常见速度线速览，展示满速、满个体、无速、减速 4 种配置的实战速度估算。
- 配队页构筑配置新增六项个体值配置，随槽位保存到 `rocom.team-builder.v1`，并兼容旧队伍数据和旧分享链接。
- `/pvp` 从已保存配队选择我方宠物时，自动读取槽位速度个体值和性格速度修正，用于实战速度线参考且不反写配队。
- `/pvp` PVP 对位助手升级到 v0.2，接入实战速度线计算，支持双方速度个体值与速度性格修正，并继续避免伤害计算和胜率预测。
- 新增 `/stats` 实战属性计算器 v0.1，并新增可复用 `src/lib/statCalculator.ts`，支持种族值、个体值和性格修正按分步四舍五入规则计算 PVP 实战属性。
- 新增 `/pvp` PVP 对位助手 v0.1，支持从已保存配队选择我方宠物、手动选择双方宠物，并展示属性倍率、速度差、种族值和轻量风险提示。
- 移除主内容滚动容器的平滑滚动效果，路由切换回顶部时改为立即跳转，统一 iPhone Safari、PWA 和桌面浏览器表现。
