# 当前版本状态

本文记录个人版 `rocom.aoe` 的当前改造状态，用于后续 Codex 新窗口快速接续。

## 当前已完成

- 首页个人版入口：首屏聚焦属性克制、图鉴搜索、技能搜索，并保留 PVP Coming Soon 与“更多工具”区域。
- 全局导航：侧边栏保留原项目功能入口，并新增正式 `/skills` 技能入口。
- 属性页优化：`/attributes` 已从圆形图谱优先改为四象限信息卡优先，图谱保留为辅助视图。
- 双属性防守面：支持两个不同属性组合的 3x、2x、0.5x、0.25x 与普通承伤分组。
- 双属性打击面：展示两个技能属性分别 2x 克制的目标，以及合并后的打击覆盖。
- 排除“首领”血脉：属性页、属性选择器、图谱、防守面和打击面均不把“首领”作为普通战斗属性。
- 独立 `/skills` 技能查询页：支持技能名称、ID、描述、属性、分类搜索与筛选。
- 技能图标与重复技能合并：`/skills` 复用 `SkillIcon.vue`，从宠物详情技能对象补充 `icon_id`；同名但不同 ID 的技能会合并，保留 `moves.json` 详情并补充 `PetSkillIndex` 来源 ID。
- PWA 第一版：新增 manifest、主屏幕图标和 service worker，支持应用壳、核心路由与核心 JSON 的基础离线缓存。

## 当前核心页面

- `/attributes`：属性克制、单属性四面、双属性防守面、双属性打击面。
- `/encyclopedia`：图鉴查询与宠物详情入口，当前原版体验暂不大改。
- `/skills`：独立技能百科查询页，第一版只查技能本身，不做完整宠物归属反查。

## 注意事项

- 本项目统一使用 Yarn：常用命令为 `yarn install`、`yarn dev`、`yarn build`。
- 不要手工修改 `public/data` 中的基础游戏数据。
- 不要修改 `scripts`，除非任务明确要求数据同步或脚本维护。
- 不要修改 `yarn.lock`，除非依赖相关任务明确要求。
- 个人改造优先放在 `src/features/*`、`src/custom` 或 `public/my-data`，降低同步上游时的冲突。
